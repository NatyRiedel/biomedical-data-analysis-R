

# Biomedical Data Analysis in R

This repository contains an example workflow for analyzing biomedical datasets using R.

## Description

The analysis includes:

- Data loading and cleaning
- Basic statistical tests (t-test / ANOVA)
- Data visualization (boxplots, scatter plots)
- Generation of a reproducible report using RMarkdown

## Tools

- R
- tidyverse
- ggplot2

## Project structure

- data/ : example dataset
- scripts/ : analysis scripts
- results/ : generated figures and tables
- report.Rmd : reproducible report
